## 使用说明

1. **启动开发服务器**: `npm start`
2. **构建生产版本**: `npm run build`
3. **在Excel中加载Add-in**: 导入manifest.xml文件
4. **配置参数**: 设置行数、起始IP、主题样式
5. **生成模板**: 点击生成按钮创建美化的IP表格
